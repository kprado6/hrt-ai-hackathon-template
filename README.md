# Mobile Recreation Park Tour

The Mobile Recreation Park Tour allows users to view the locations, dates, and times where the City of Clovis Recreation Department will bring its mobile recreation trailer.

## Features

1. Live map allows users to navigate to park locations

2. Organizes locations by month
3. Displays event time frames for each location

## Purpose

This project was created to help the public to navigate which parks free recreation is offered in local parks. The pinned locations help users easily view recreation event locations, schedules and details through an interactive map. 



## Usage
1. Open the application - The dashboard loads with an interactive event map and an event schedule

2. Select filter - Each event can be filtered by month
3. View Mobile Recreation locations - Locations appear based on filter selected
4. Select a location for more details - Each location displays the park name, the date of the event, the time and the address of the park
